# Trading Strategy Analysis Dashboard

Interactive dashboard for comprehensive trading strategy performance analysis and risk assessment using Python and Jupyter widgets.

## Features

- **Interactive Visualizations**: Dynamic equity curves, drawdown analysis, and profit/loss distributions
- **Trade Analysis**: Performance breakdown by hour, direction, and exit reasons  
- **Risk Metrics**: Value at Risk (VaR), Expected Shortfall, and Sharpe ratio calculations
- **Monte Carlo Simulations**: Strategy robustness testing with slippage modeling
- **Performance Analytics**: Win rate, profit factor, and maximum drawdown analysis

## Technical Architecture
Data Loading → Performance Metrics → Interactive Dashboards → Risk Analysis → Strategy Assessment

## Dashboard Components

- **Performance Overview**: Equity curves, drawdown analysis, rolling performance metrics
- **Trade Analytics**: Hourly performance, directional analysis, exit reason breakdown
- **Risk Management**: VaR calculations, Monte Carlo simulations, risk-adjusted returns
- **Strategy Assessment**: Automated scoring and recommendations based on risk tolerance

## Analysis Metrics

- **Returns**: Total return, net profit, profit factor
- **Risk**: Sharpe ratio, maximum drawdown, Value at Risk
- **Performance**: Win rate, average trade, trade duration analysis
- **Simulation**: Probability of profit, expected shortfall, confidence intervals
